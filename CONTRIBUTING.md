# Standards
This software is made to be simple to use and install. Please do not add things that require dependencies, as this would make it less accessible. Other than that, any change can be made that makes this app a more functional UML diagram generator. 

# Drawio files
This software creates an export file in the .drawio format. This format uses XML, a markup language which will become visible on the website drawio.com. Changes made to this software are required to keep this format for exports. Other export formats may be added, but theyare not allowed to replace.
